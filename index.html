Projects

  • ScholarAI

    This project a research paper assistant with a multi-agent workflow that helps users search arXiv, generate summaries, draft literature reviews, create citations, and manage saved paper collections through a simple web interface. The project combined a React frontend, FastAPI backend, a remote summary agent, and Azure Blob Storage to support academic research.

  • Amazon Product Search

    The multimodal product search system was built on the Amazon Berkeley Objects dataset, using a fine-tuned SigLIP model and FAISS to support efficient text-to-image and image-to-text retrieval. The system improved alignment between product images and descriptions, enabling more accurate semantic search and a scalable foundation for e-commerce discovery.

  • Amazon Product Reviews

    Analyzing product feedback manually is challenging due to the high volume and unstructured nature of reviews. Here, we use K-means to cluster Amazon appliance reviews and ChatGPT-4 to label and summarize each cluster. This approach provides clear, structured insights, reducing information overload and enhancing feedback analysis.

  • Airbnb Fraud Detection

    This project aims to develop and deploy an Airbnb fraud detection program using a Feed-Forward Neural Network (FNN) trained on the Boston Airbnb Fraud Detection dataset. For scalability and deployment, we use Docker, Kubernetes, and Azure Cloud, ensuring reliable fraud detection with deep learning and cloud computing.

  • Movie Reviews Sentiment

    This project focuses on building robust classification models for sentiment prediction using the Kaggle movie review dataset. Implement and compare the performance of Naïve Bayes, Random Forest, Multi-Layer Perceptron (MLP), and Recurrent Neural Network (RNN) classifiers.

  • Hotel Database Management

    We developed a Database Management System (DBMS) to facilitate business operations for a fictitious hotel, Ampezzo Regency, located in a tourist-rich region in Italy. The database was built using Microsoft SQL Server and implemented in PowerApps for efficient data management and accessibility.
